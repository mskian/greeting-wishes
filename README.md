## Setup

- Download or Clone the repo
- install dependencies

```sh
pnpm install
```

- Development

```sh
pnpm dev
```

- Build a Project

```sh
pnpm build
```

- Start the server

```sh
pnpm start
```

## Usage

- `/` : input form for create Greeting
- `/your-name` - Directly create a greeting page  

```sh

## Home page
http://localhost:6020/
```
