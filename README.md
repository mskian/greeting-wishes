## Setup

- Download or Clone the repo

```sh
git clone --branch termux https://github.com/mskian/greeting-wishes/
```

- install dependencies

```sh
pnpm install
```

- Development

```sh
pnpm dev
```

- Build a Project

```sh
pnpm build
```

- Start the server

```sh
pnpm start
```

## Usage

- `/` : input form for create Greeting
- `/your-name` - Directly create a greeting page  

